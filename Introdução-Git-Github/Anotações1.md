# Git e Github Revisão
Trazendo repositório para máquina local:
Dentro da pasta desejada, digite: git clone + (Link ssh)
Entrando no repositório: cd (caminho da pasta)
Adicionando arquivos para o git: git add . ou git add -A
Commit: git commit -m "Inclusão das anotações do curso de Git/Github"
Por fim push origin main e atualize com f5 no git que vai aparecer tudo.
